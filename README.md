🖼️ Gallery App - Backend API
A robust RESTful API built with Node.js and Express.js, designed to manage a digital image gallery with a focus on clean architecture and scalable data handling. This project demonstrates my proficiency in building secure backends and integrating NoSQL databases.

🛠️ Tech Stack & Skills
Runtime Environment: Node.js.

Framework: Express.js.

Database: MongoDB with Mongoose ODM for flexible data modeling.


Environment Management: Secure configuration using dotenv for sensitive credentials like JWT_SECRET and DB_URL.

Architecture: Implementation of the MVC (Model-View-Controller) design pattern.

🚀 Key Features
Full CRUD Functionality: Developed endpoints to Create, Read, Update, and Delete image records seamlessly.

Image Upload Logic: Integrated logic to handle image file metadata and storage.

Error Handling: Implemented try-catch blocks to ensure API stability and professional response management.

Database Integration: Established a secure connection to MongoDB with automated error logging.

📁 Project Structure
Controllers: Contains the core business logic for image management.

Models: Defines the Schema for image data, including titles and filenames.

Routes: Organized routing system to maintain clean and navigable code.
